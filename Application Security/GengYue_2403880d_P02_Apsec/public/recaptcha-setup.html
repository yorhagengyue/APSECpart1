Google reCAPTCHA Setup Guide

Current Status: Using test keys (always passes verification)

What is reCAPTCHA?

Google reCAPTCHA is a free service that protects your website from spam and abuse. It uses advanced risk analysis techniques to tell humans and bots apart.

TechSecure has integrated reCAPTCHA v2 ("I'm not a robot" checkbox) on:

  • User Registration Page
  • User Login Page

How to Set Up Production Keys

  1. Visit Google reCAPTCHA Admin Console
    Go to https://www.google.com/recaptcha/admin
  2. Register Your Site
    • Label: TechSecure (or your site name)
    • reCAPTCHA type: reCAPTCHA v2
    • Select: "I'm not a robot" Checkbox
    • Domains: Add your production domain(s)
  3. Get Your Keys
    After registration, you'll receive:
    • Site Key: For the frontend (public)
    • Secret Key: For the backend (private)
  4. Update Environment Variables
    Add to your .env file: 
    RECAPTCHA_SITE_KEY=your-site-key-here
RECAPTCHA_SECRET_KEY=your-secret-key-here
  5. Update Frontend HTML
    Replace the test site key in login.html and register.html: 
    <div class="g-recaptcha" data-sitekey="your-site-key-here"></div>

Testing reCAPTCHA

To test the reCAPTCHA integration:

  1. Go to the Registration or Login page
  2. Fill in the form
  3. Click the "I'm not a robot" checkbox
  4. Submit the form

With test keys, the checkbox will always pass. With production keys, users may need to solve a challenge.
