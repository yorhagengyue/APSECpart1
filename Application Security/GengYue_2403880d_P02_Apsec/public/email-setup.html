Email Verification Setup Guide

Current Email: tpmswebservice@gmail.com

What You Need

To enable email verification, you need:

  1. Gmail Account: tpmswebservice@gmail.com
  2. App-Specific Password: NOT your regular Gmail password
  3. 2-Step Verification: Must be enabled on your Gmail account

Step 1: Enable 2-Step Verification

  1. Go to Google Account Security
  2. Click on "2-Step Verification"
  3. Follow the setup process
  4. Enable 2-Step Verification

Step 2: Generate App Password

  1. Go to App Passwords
  2. Select "Mail" as the app
  3. Select "Other" as the device and enter "TechSecure"
  4. Click "Generate"
  5. Copy the 16-character password (spaces don't matter)
Important: The app password is shown only once. Save it securely!

Step 3: Configure Environment Variables

Add these to your .env file:

# Email Configuration
EMAIL_USER=tpmswebservice@gmail.com
EMAIL_APP_PASSWORD=your-16-character-app-password-here

Example:

EMAIL_APP_PASSWORD=abcd efgh ijkl mnop
Security: Never commit your .env file to version control!

Testing Email Configuration

When you start the server, you'll see one of these messages:

  • ✅ Email server is ready to send messages - Configuration is correct!
  • ❌ Email configuration error - Check your app password

To test email sending:

  1. Register a new account
  2. Check the email inbox for verification email
  3. Click the verification link
  4. Confirm email is verified

Email Features

1. Registration Email
  • Sent immediately after registration
  • Contains verification link
  • Valid for 24 hours
2. Verification Success Email
  • Sent after successful verification
  • Confirms account is active
  • Contains login link
3. Resend Verification
  • Available if original email not received
  • Can be triggered from login page
  • Generates new verification token

Troubleshooting

Email not sending?
  • Check app password is correct (16 characters)
  • Ensure 2-Step Verification is enabled
  • Check server console for error messages
  • Verify EMAIL_APP_PASSWORD is in .env file
Email in spam folder?
  • Add tpmswebservice@gmail.com to contacts
  • Mark as "Not Spam" if found there
  • Check email filters
Common Errors:
  • "Invalid login" - Wrong app password
  • "Please log in via your web browser" - Need app password, not regular password
  • "Connection timeout" - Check firewall/network settings
